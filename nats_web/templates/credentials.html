Required IAM Permissions

Your AWS credentials need the following permissions:

  • lambda:ListFunctions - To fetch available Lambda functions
  • lambda:InvokeFunction - To trigger Lambda functions (if needed)
Your AWS Access Key ID (starts with AKIA)
Your AWS Secret Access Key (keep this secure)
AWS region where your Lambda functions are deployed
Tip: You can create IAM credentials specifically for this application with minimal required permissions.
